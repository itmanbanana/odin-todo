To-Do App Brainstorming:

- To-do project exist as cards that are appended to the end of a main project
list. Individual items are appended to each project as smaller cards/list items.
- Each to-do item card has the following properties:
    - Title
    - Description (limit characters?)
    - Due date
    - Priority
    - Assigned to a project
- Need forms for creating new projects and project list items

- Objects/classes:
    - ToDoManager:
        - Properties:
            - List of ToDoProject objects
        - Create new ToDoProject
        - Delete project (exposed as public to be called by ToDoProject)
        - Modify ToDoProject(title, desc)
    - ToDoProject:
        - Properties:
            - Title (name)
            - Description
            - List of ToDoItem objects
        - Self deletion via ToDoManager
        - Self modification via ToDoManager
        - Create new ToDoItem (to go under self)
        - Delete ToDoItem (exposed as public to be called by ToDoItem)
        - Modify ToDoItem(title, desc, duedate, priority)
            - Reassign ToDoItem to new passed params
    - ToDoItem:
        - Properties:
            - Project assigned => reference to ToDoProject
            - Title
            - Description
            - Due date
            - Priority
            - Completed (boolean) - checkbox
        - Self deletion via ToDoProject
        - Self modification via ToDoProject
    - DisplayManager:
        - Update display: iterates through all ToDoProject and respective
        ToDoItem objects in ToDoManager for DOM modification (may require
        helper functions)
        - Forms triggering for creation/modification of projects/items
        - Event listeners for project/item creation and deletion (buttons?)

- ToDoProject HTML Layout:
    - div.project (ToDoProject)
        - div.project-title
        - div.project-desc
        - div.project-settings
            - button.project-modify (change title/desc)
            - button.project-delete (delete this project)
            - button.add-item (adds ToDoItem to this project)
        - ul.item-list
            - li.item (ToDoItem)
            - li.item (ToDoItem)
            - li.item (ToDoItem)
            ...

- ToDoItem HTML Layout:
    - li.item (ToDoItem)
        - div.item-title
        - div.item-desc
        - div.item-date
        - div.item-priority
        - input[type="checkbox"].item-completed
        - div.item-settings
            - button.item-modify (change title/desc/dueDate/priority)
            - button.item-delete (delete this item)
